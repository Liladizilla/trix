What's actually built

No vague claims — every item below compiles, links, and runs.

01
Boot & interrupts done
GRUB2 multiboot2, GDT with ring 0/3 descriptors, IDT, PIC, timer, keyboard driver
02
Memory management done
PMM (physical memory manager), VMM (virtual memory, 4-level paging), slab heap allocator
03
System calls done
SYSCALL/SYSRET (AMD64 fast path), capability-based security model, userspace ring 3 entry
04
Filesystem & ELF loader done
VFS abstraction layer, FAT32 driver (read/write), ELF64 binary loader with dynamic linking stubs
05
Framebuffer graphics done
UEFI/BIOS framebuffer driver, PSF2 bitmap font renderer, filled primitives, double-buffering
06
Window manager in progress
Framebuffer-backed WM, desktop shell, application launcher, Z-order management
07
IPC & networking in progress
Message-passing IPC, UNIX-style sockets, userspace socket API
